App name: VPN Manager
Design: modern dark mode app with indigo/purple style, rounded cards, clean iOS look.

Main features:
1. Customer list screen
- Show all customers in cards.
- Each card shows:
  - customer name
  - monthly price/payment amount
  - user count, like “20 users”
  - username/login
  - password
  - next payment date
  - status: Active, Expired, Not Extended
- Sort customers by nearest payment date.
- Highlight overdue payments in red.
- Highlight payments due today/tomorrow in orange/indigo.

2. Add customer
Fields:
- Customer name
- Payment amount
- Currency: USDT / USD / Manat / other
- User count
- VPN panel username
- VPN panel password
- Notes
- Payment date
- Monthly billing toggle

3. Customer detail page
Buttons:
- Extended
- Didn’t Extend
- Edit
- Delete
- Copy username
- Copy password

4. Extended button

- Move payment date forward by 1 month automatically.
- Mark customer as Active.
- Save payment history with date and amount.

5. Didn’t Extend button
When I tap it:
- Mark customer as Not Extended/Expired.
- Keep customer in list but show red status.

6. Notifications
- Send local iOS notifications when customer payment date arrives.
- Also remind me 1 day before.
- Notification text example:
  “Payment due: Merdan — 400m — 20 users”
- Ask notification permission on first launch.
- Notifications must update when payment date changes.

7. Security

- Use secure storage/encryption for passwords if possible, preferably iOS Keychain.
- Do not use any external server.
- Add Face ID / passcode lock before opening the app.

8. Data

- Include search by customer name.
- Include filters: All, Active, Due Soon, Expired, Not Extended.
- Add backup/export to JSON and import from JSON.

9. UI screens
